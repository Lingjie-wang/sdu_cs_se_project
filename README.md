## 项目结构

使用命令：

```shell
treer -i ".git" ## 忽略 .git 文件夹
```

> 参考：
>
> [markdown如何写出项目目录结构 - 简书](https://www.jianshu.com/p/e38a07f824a2)

### 目录树如下
概况：
```shell
├─README.md 
├─img ## 存放图片
├─doc ## 存放项目文档
```
详细：

```shell
├─README.md
├─img
├─doc
|   ├─lab4
|   ├─lab3
|   ├─lab2
|   ├─lab1
```
